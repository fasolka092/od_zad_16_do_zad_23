Pojęcie Znaczenie pojęcia uzupełnia uczeń
Co znaczy skrót ID? To skrót od "identifier", czyli identyfikator.
Jaki jest cel stosowania ID? Służy do unikalnego oznaczenia jednego konkretnego elementu na stronie, by nadać mu indywidualne style lub go zidentyfikować w skryptach.
Jakie jest znaczenie znaku # Znak # w CSS oznacza, że odnosimy się do elementu z konkretnym identyfikatorem (ID). Np. #mojeID oznacza element z id="mojeID".
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W HTML: <p id="mojeID">Tekst</p>
W CSS: #mojeID { kolor: czerwony; } – stosujemy znak # przed nazwą identyfikatora, by przypisać styl.
Uwagi na temat stosowania ID (trzy konieczne wyliczenia). 1. ID musi być unikalne – nie może się powtarzać w jednym dokumencie.
2. Stosuje się tylko jeden ID na dany element.
3. Nazwa ID nie może zawierać spacji i powinna zaczynać się literą.
